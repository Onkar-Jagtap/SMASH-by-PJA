# S.M.A.S.H. MATCHER 🥊🍄
**(Company Match Intelligence Platform vs Gemini 1.5 Flash)**

S.M.A.S.H. Matcher is an enterprise-grade Corporate Entity Resolution tool disguised as an aggressive 8-bit retro arcade game. It safely, accurately, and rapidly resolves variations of corporate identities, crushing lists of up to 500 targets against suppression grids spanning 100k+ rows! 

## 🔥 Key Enterprise Features
1. **The Fuzz Engine**: Custom `O(1)` string inverted-index algorithms locally crush millions of Levenshtein string variants instantly without crashing your browser.
2. **AI Arena (Gemini 1.5 Flash)**: When mathematical heuristics are "murky" (e.g. comparing "Global IT" vs "Global Technologies"), the algorithm funnels the dispute to a lazy-loaded Express proxy connected to Gemini Flash.
3. **Smart Quota Gates**: Conserves API budgets by rejecting garbage completely out of bounds and auto-matching mathematically flawless hits, isolating AI power for only the toughest edge cases.
4. **Exponential API Armor**: Rate limits (HTTP 429) don't crash this app! S.M.A.S.H. silently absorbs timeouts and seamlessly trickles AI API retries at back-offs of 2s, 4s, 8s, 16s, and 32s!
5. **Excel / CSV Loot Drops**: Export battle logs natively to strictly formatted `.xlsx` or `.csv` spreadsheets, fully retaining the savage, 1-sentence analytical decisions generated by the Announcer Bot!

## 🕹️ Gamified Workflows
Why stare at a loading spinner when you can squash Alien Bugbots?
While S.M.A.S.H. MATCHER processes the AI queue, a complete reflex clicker Mini-Game activates directly in the HUD allowing you to score points while you wait! 

## 🛠️ Stack & Installation
- **Frontend Core:** React, Vite, TailwindCSS (Retro Customized `index.css`)
- **Backend Architecture:** Express `server.ts` proxy for secured AI interactions.
- **AI Model:** `@google/genai` (Gemini 1.5 Flash).
